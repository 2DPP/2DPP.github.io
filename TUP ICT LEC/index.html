Introduction to CSS

CSS (Cascading Style Sheets) is a language used to style HTML documents. It describes how HTML elements should be displayed on screen, paper, or in other media.

Importance of CSS

  • Separation of content from design.
  • Enhances web accessibility.
  • Faster website load times.
  • Easier maintenance across multiple pages.

Selectors

CSS selectors are used to target HTML elements based on their attributes, id, class, or type.

h1 { color: orange; }

This is a header with font color: orange

Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree.

Embedding multimedia

Embedded media definition: embedded media refers to multimedia items that are placed inside a web page from third-party or off-site resources.

Multimedias to be embedded:

  • Images/Gifs
  • Videos
  • Audios
  • Videogames
